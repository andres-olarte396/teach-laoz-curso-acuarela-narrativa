# Curso: Pintura con Acuarela y Narrativa "Colores para el Alma"

## Descripción General

Curso terapéutico y recreativo diseñado para adultos mayores. Experimenta el placer de mezclar agua y color, reduciendo la ansiedad y estimulando la memoria a través de la narrativa personal.

- **Nivel**: Básico
- **Duración**: 10 Horas
