# EVALUACIÓN: MI PAISAJE INTERIOR

**Instrucciones**: Contempla tu horizonte.

1. [ ] ¿Esperaste que secara antes de quitar la cinta? (Si se rompió un poquito el papel, no pasa nada, es experiencia).
2. [ ] ¿Se nota la diferencia entre el cielo y el mar?
3. [ ] ¿Te transmite calma ver esa línea recta?
4. [ ] ¿Fue satisfactorio quitar la cinta y ver el borde limpio?

*El horizonte siempre está ahí para orientarnos.*
