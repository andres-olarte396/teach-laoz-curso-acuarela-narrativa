# EVALUACIÓN: MI EQUIPAJE

**Instrucciones**: Verifica que tienes lo esencial.

1. [ ] Tengo papel grueso (especial acuarela o cartulina gruesa).
2. [ ] Tengo un pincel que termina en punta redonda.
3. [ ] He llenado dos recipientes con agua.
4. [ ] Tengo un paño o servilleta a mano para secar.
5. [ ] He tocado el papel y entiendo que tiene dos caras (una más rugosa que otra, usaremos la rugosa). *Nota: Si ambas son iguales, no importa.*
