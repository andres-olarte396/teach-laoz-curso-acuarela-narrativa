# EVALUACIÓN: LISTA DE CHEQUEO

**Instrucciones**: Marca con una X si has completado cada paso. No hay notas, solo la satisfacción de estar listo.

1. [ ] He encontrado un lugar con buena luz que no cansa mis ojos.
2. [ ] Mi mano no hace sombra sobre el papel al escribir o pintar.
3. [ ] He retirado de la mesa todo lo que no voy a usar para pintar.
4. [ ] Me siento cómodo/a en la silla y mis pies tocan el suelo (o un apoyo).
5. [ ] Tengo mis materiales (agua y pinturas) del lado de mi mano dominante.
